# MERN Stack E-Commerce Web Application

A full-stack e-commerce web application built using the **MERN stack**, designed to provide a complete online shopping experience with modern UI and scalable backend architecture.

-------------------------------------------

## Overview
This project is a fully functional e-commerce platform that allows users to browse products, manage a shopping cart, and place orders. It demonstrates **end-to-end full stack development**, including frontend UI, backend APIs, database integration, and authentication.

The goal of this project was to build a real-world application that reflects industry-level workflows and best practices.

## Features

### User Features
- User authentication (Sign up / Login)
- Browse products with details
- Add / remove products from cart
- Order placement functionality
- Responsive design for all devices

### Admin / Management Features
- Product management (Add / Update / Delete products)
- Secure API endpoints
- Structured REST API architecture
